## BookingFlow — nuværende tilstand

### Trin-rækkefølge
```
lifttype → boat → timeslot → info → summary → confirmed
```

### State-interface
```typescript
interface State {
  liftType:           'kun-kranløft' | 'vinteropbevaring' | 'bedding' | null
  boatType:           'sejlbåd' | 'motorbåd' | null
  mast:               'med-mast' | 'uden-mast' | null
  wantsStormstøtter:  boolean | null
  weight:             '0-3t' | '3-6t' | '6-12t' | '12-20t' | null
  timeSlot:           TimeSlot | null
  boatName:           string
  boatWidth:          string
  ownerName:          string
  email:              string
  phone:              string
}
```

### Layout (desktop)
- **3-kolonne grid**, `items-stretch` — venstre 2 kolonner = formular, højre 1 kolonne = panel
- **Højre kolonne**: `flex flex-col` med:
  - `PricePanel` øverst (`sticky top-6`) — viser løbende prisberegning
  - "Fortsæt"-knap med `mt-auto` — flyder til bunden, ingen boks rundt om
- **"← Tilbage"** sidder øverst i venstre kolonne (vises alle trin undtagen lifttype)
- Mobile: højre kolonne skjult, "Fortsæt" vises inline i venstre kolonne

### Komponenter

**`ChoiceCard`** — soft look:
- `rounded-xl`, `border` (1px), `px-4 py-3.5`
- Valgt: `border-rust bg-rust/5 shadow-sm` + rust checkmark-cirkel øverst til højre
- Tekst forbliver charcoal/rust (ikke hvid), label bliver rust ved valg

**`FieldCheck`** — rust cirkel (w-4 h-4) med hvidt SVG-hak. Bruges:
- I label ud for "Bådens navn" når udfyldt
- I label ud for "Bredde" når udfyldt
- I label ud for "Vægt" når valgt
- Absolut positioneret øverst til højre i valgt vægt-knap

**`PricePanel`** — sticky højre panel:
- Viser linjeposter efterhånden som valg træffes
- Inkluderer transport (473 kr.) ved vinteropbevaring
- Stormstøtter: inkluderes automatisk ved sejlbåd + med mast + opbevaring

**`Trail`** — pill-badges øverst som opsummerer valgte trin

### Validering — "Fortsæt" disabled når:
```typescript
// Boat-trin:
!s.boatType || !s.weight || (
  s.boatType === 'sejlbåd' && (
    !s.mast || (s.mast === 'uden-mast' && s.wantsStormstøtter === null)
  )
)

// Info-trin:
!s.ownerName || !s.email || !s.phone
```

## AdminPanel — nuværende tilstand

### Tre faner

**Bookinger**
- Tabel over alle bookinger
- Status-skift via `<select>` dropdown (bekræftet / afventer / annulleret)
- Inline slet-bekræftelse (to-trins: "Slet?" → "Ja / Nej")
- Filter-knapper: alle / stativleje / kranløft / stormstøtter

**Lager**
- Stativpladser med belægningsbars
- Inline rediger (betegnelse, total, booket)
- "+ Nyt stativ" — dashed border add-form nederst
- Inline slet-bekræftelse

**Tidsvinduer**
- Slots grupperet per dato med dato-header i charcoal
- **Batch-oprettelse** med to modes:
  - *Enkelt dag*: vælg dato + tilføj tidspunkter + kapacitet
  - *Uge-skabelon*: vælg ugedage (Man–Søn), tilføj tidspunkter, sæt fra/til dato, kapacitet
  - Live-tæller: "→ Opretter X tidsvinduer"
  - Duplikater springes automatisk over
- Inline rediger per slot (dato, tid, kapacitet)
- Inline slet-bekræftelse

## Design-tokens (Tailwind `@theme`)
- `charcoal` — mørk næsten-sort (primær tekst/baggrund)
- `charcoal-mid` — lidt lysere charcoal (hover-states)
- `rust` — rødlig accent (#c0392b-agtig)
- `rust-dark` — mørkere rust (hover)
- `rust-light` — lysere rust (hero-tekst)
- `warm-gray` — borders og skillelinjer
- `offwhite` — sidebaggrundsfarve
